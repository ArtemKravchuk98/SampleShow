Постійний запам'ятовувач, пам'ять постійного зберігання, постійний запам'ятовувальний пристрій, пам'ять тільки для читання (англ. ROM - Read Only Memory) — носій інформації, що використовується тільки для отримання і відтворення даних. Зазвичай інформація записується на такий носій в заводських умовах, недоступних споживачу. Зазвичай ROM використовується для керування роботою апаратної частини комп'ютера. Найважливішою мікросхемою ROM комп'ютера є BIOS (Basic Input/Output System — базова система введення/виведення). Комп'ютер звертається до BIOS зразу ж після подачі живлення на центральний процесор, ще до завантаження операційної системи. Мікросхема ППЗ здатна тривалий час зберігати інформацію, навіть при вимкненому комп'ютері. Кажуть, що програми, які знаходяться в ППЗ, «зашиті» у ній — вони записуються туди на етапі виготовлення мікросхеми. Комплект програм, що знаходиться у сучасних ППЗ, утворює базову систему введення/виведення BIOS (Basic Input Output System). Основне призначення цих програм полягає в тому, щоб перевірити склад та працездатність системи та забезпечити взаємодію з клавіатурою, монітором, жорсткими та гнучкими дисками.
